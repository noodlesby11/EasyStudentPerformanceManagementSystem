# 班级学生成绩管理系统

## 介绍

简单的班级学生成绩管理系统，利用文件保存学生信息而**不是数据库**

## 主要功能

对学生信息的增、删、改、查

## 各个文件的作用

* add.c 实现添加学生信息的功能
* DeleteDoe.c 删除学生信息的功能，指删除全部信息~~非常的粗暴~~
* fread.c 进行文件的读取，用自己的文件可以修改一下路径
* modify.c 实现修改学生信息的功能
* savef.c 保存文件
* student.h 定义了学生的结构
* StudentInfo.txt 学生的个人信息
* StudentScore.txt 学生的成绩信息
